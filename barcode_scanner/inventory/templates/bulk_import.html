Import Instructions
  1. Download the appropriate template for the asset type you want to import.
  2. Fill in the template with your asset data. The first row contains column headers - do not modify them.
  3. Save the file as a CSV (Comma Separated Values) file.
  4. Select the asset type and upload the CSV file.
  5. The system will validate and import the data. Existing assets (matched by Asset Tag or Service Tag) will be updated.
Note: Date fields should be in YYYY-MM-DD format (e.g., 2024-01-15). Status should be one of: active, retired, repair, disposed, available.
